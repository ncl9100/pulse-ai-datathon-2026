06_ROOT_CAUSE
=============

Five cause-and-effect chains diagnosed per project, with estimated dollar impacts.
  cause_effect_diagnosis.csv      — Master table: all 5 chain scores per project,
                                    total estimated loss, primary cause label
  chain_A_design_rework.csv       — Scope Gap + Design Error CO rework labor cost
  chain_B_material_idle.csv       — Partial shipment idle labor + OT acceleration premium
  chain_C_rfi_standby.csv         — Late RFI responses x daily crew cost = standby loss
  chain_D_rejected_co_loss.csv    — Full value of rejected positive-amount COs
  chain_E_early_cos.csv           — CO count in first 20% of project (design risk signal)
