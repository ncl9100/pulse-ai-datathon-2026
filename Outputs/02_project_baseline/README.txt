PROJECT BASELINE
=================
One row per project. Built from contracts + SOV budget + change orders.

Key columns:
  original_contract_value  : signed contract price
  total_estimated_cost     : sum of all estimated labor + material
  builtin_margin_usd/pct   : how much profit margin was in the original price
  revised_contract_value   : original + all approved change orders
  co_rejection_rate_pct    : % of change orders rejected by the GC
  project_type             : Small / Medium / Large / Mega
