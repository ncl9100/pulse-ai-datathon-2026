# pulse-ai-datathon-2026
Pulse AI Datathon Repo hosted by DSC Club @NYU

## Group name
BNKT Margin Guardians

## Stack
v0 + Next.js + Shadcn/ui + Recharts + Claude/OpenAI API + DuckDB

## Overview

This project builds an autonomous AI agent that analyzes a portfolio of construction projects, identifies margin erosion, explains root causes, and recommends concrete recovery actions.

Unlike traditional dashboards, our system:
- Scans the entire portfolio automatically
- Flags at-risk projects without prompts
- Diagnoses why margin is eroding
- Suggests specific, dollar-impact actions
